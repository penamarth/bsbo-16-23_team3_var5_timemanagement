# Диаграмма последовательностей UML для сценария "Учет выполения задач"
![Диаграмма последовательностей](../../.images/track-Task-Executionaction.png)

```
@startuml
actor "Руководитель" as Manager
participant "TaskManagement" as Controller
participant "IUserStorage" as UserStorage
participant "Task[] (in-memory)" as TaskArray
participant "ReportGenerator" as ReportGen
participant "Бухгалтерия" as Accounting

Manager -> Controller: TrackTaskExecution(managerId)
activate Controller

Controller -> UserStorage: GetById(managerId)
UserStorage --> Controller: User

Controller -> TaskArray: EnumerateTasks()
activate TaskArray
TaskArray --> Controller: текущие статусы + workload
deactivate TaskArray

Controller -> ReportGen: generateExecutionReport(tasksSummary)
activate ReportGen
ReportGen --> Controller: executionReport
deactivate ReportGen

Controller -> Accounting: sendPayrollData(tasksSummary)
activate Accounting
Accounting --> Controller: dataReceived
deactivate Accounting

Controller -> Manager: ShowReport(executionReport)
deactivate Controller
@enduml
```
