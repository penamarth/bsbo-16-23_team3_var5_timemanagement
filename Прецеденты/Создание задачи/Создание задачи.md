# Диаграмма последовательностей UML для сценария "Создание задачи"
![Диаграмма последовательностей](../../.images/create-task.png)

```
@startuml
actor "Руководитель" as Manager
participant "TaskManagement" as Controller
participant "IUserStorage" as UserStorage
participant "Task" as TaskObj
participant "NewTaskState" as State

Manager -> Controller: CreateTask(managerId, title, description)
activate Controller

Controller -> UserStorage: GetById(managerId)
activate UserStorage
UserStorage --> Controller: User
deactivate UserStorage

Controller -> TaskObj: new Task(id, title, description, Controller)
activate TaskObj
TaskObj -> State: SetState(NewTaskState)
deactivate State
TaskObj --> Controller: createdTask
deactivate TaskObj

Controller -> Manager: TaskCreated(taskId)
deactivate Controller
@enduml
```
